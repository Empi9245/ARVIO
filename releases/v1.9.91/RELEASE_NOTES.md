# ARVIO 1.9.91

## Key changes
- Improved IPTV categories, context actions, mobile fullscreen TV playback, and IPTV VOD quality selection. Contributors: ARVIO maintainers.
- Fixed Trakt watchlist ordering/matching, watchlist empty-state regressions, and faster Continue Watching startup. Contributors: ARVIO maintainers.
- Fixed watchlist topbar down-navigation so focus returns to the grid correctly. Contributor: EierkopZA; MR submitted by EierkopZA.
- Improved source switching, playback controls, stream startup, trailer handling, loading clearlogo polish, and source loading from fast Search-to-Details navigation. Contributors: EierkopZA, Himanth Reddy, and ARVIO maintainers; MR submitted by EierkopZA.
- Added catalog list discovery for Trakt/MDBList and improved catalog focus/navigation stability. Contributors: ARVIO maintainers, Himanth Reddy, and silentbil.
- Improved app language coverage, subtitle language filtering, profile loading, DNS persistence, and cloud login/startup language restore. Contributors: silentbil, Himanth Reddy, EierkopZA, and ARVIO maintainers.
- Fixed anime episode source matching, details metadata behavior, Crunchyroll assets, and several layout/focus regressions. Contributors: ARVIO maintainers, EierkopZA, and Himanth Reddy.

## Contributors
- EierkopZA: player source switching, loading clearlogo polish, regex filter TV focus, details metadata behavior, watchlist topbar/grid focus, IMDb-ID wait for faster Details source reliability, and submitted the related watchlist/source-loading MRs.

- ARVIO maintainers: IPTV category/sidebar work, VOD quality variants, watchlist/Trakt fixes, continue watching, anime episode matching, catalog discovery, cloud login/language restore, mobile fullscreen TV.
