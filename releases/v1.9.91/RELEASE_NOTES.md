# ARVIO 1.9.91

## Watchlist and continue watching
- Fixed watchlist empty state regressions and dialog language.
- Fixed watchlist loading and catalog discovery focus issues.
- Show cached continue watching on startup so the row appears immediately.
- Fixed Trakt watchlist sync and matching.

## IPTV and TV improvements
- Improved IPTV category sidebar behavior and layout.
- Merged latest TV and IPTV smooth sweep and sync fixes.

## Catalog and home screen
- Added catalog list discovery.
- Improved catalog discovery add flow.
- Fixed lag on the main screen caused by Top 10 rows.

## Player and subtitles
- Redesigned subtitle menu as a two-panel language/track layout.
- Added subtitle language filter and exposed it in mobile settings.
- Fixed player focus issues.
- Added volume control to TrailerPlayer.
- Refactored TrailerPlayer to use dependency injection for the YouTube extractor.

## Profile and settings
- Updated skip profile selection logic.
- Refactored active profile loading state management.

## Services and metadata
- Updated Crunchyroll service assets with new cover image and hero video URL.

## Stability
- Added Sentry crash reporter for improved error tracking.
